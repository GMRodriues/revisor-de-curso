# Fase 2: Revisao De Pagina

## Objetivo

Revisar cada pagina/slide do curso com verificacoes objetivas.

## Verificacoes Base

- pagina carregou
- URL/indice corresponde ao manifest
- texto principal visivel
- imagens carregaram
- videos embutidos foram detectados
- nao houve erro critico de console
- nao houve falha de rede relevante
- navegacao anterior/proximo funciona
- nao ha overflow ou sobreposicao obvia

## Saidas

- evidencia por pagina
- achados classificados
- paginas aprovadas
- paginas bloqueadas
- itens para revisao humana

## Observacao

Verificacoes visuais devem ser calibradas com exemplos reais. No inicio, preferir detectar problemas obvios e registrar `manual_review` para julgamentos subjetivos.
