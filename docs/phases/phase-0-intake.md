# Fase 0: Entrada

## Objetivo

Coletar informacoes minimas para uma revisao automatizada segura.

## Entradas

- URL da plataforma
- forma de login ou instrucao para usar perfil ja logado
- curso ou lista de cursos
- exemplos de revisao humana esperada
- criterios obrigatorios de lancamento
- viewports alvo
- limite de profundidade da revisao

## Saidas

- escopo congelado
- riscos conhecidos
- manifest inicial ou plano de descoberta
- criterios selecionados
- decisoes sobre evidencias

## Checklist

- [ ] Identificar plataforma e ambiente
- [ ] Definir credenciais/perfil de navegacao
- [ ] Definir curso alvo
- [ ] Definir viewports
- [ ] Definir criterios obrigatorios
- [ ] Definir formato de relatorio
- [ ] Confirmar o que fica fora do escopo
