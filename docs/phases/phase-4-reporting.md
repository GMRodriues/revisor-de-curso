# Fase 4: Relatorio

## Objetivo

Transformar evidencia coletada em um relatorio acionavel.

## Relatorio Executivo

- curso
- escopo revisado
- cobertura
- estado sugerido
- bloqueios
- principais riscos
- itens manuais pendentes

## Tabela De Achados

Cada achado deve conter:

- id
- severidade
- categoria
- localizacao
- esperado
- observado
- evidencia
- reproducao
- recomendacao
- estado

## Estado Do Curso

- `ready_candidate`
- `ready_with_minor_notes`
- `needs_fixes`
- `blocked`
- `manual_review_required`
