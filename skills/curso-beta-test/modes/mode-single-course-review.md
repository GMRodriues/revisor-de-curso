# Modo: Revisao De Curso Unico

## Objetivo

Revisar um curso, modulo ou aula com escopo fechado.

Para escopos INA/NAU, preferir `ina-nau-course-review`. Quando este modo for usado, ainda assim aplicar captura unica por pagina quando a revisao exigir navegar pagina a pagina.

## Entradas

- manifest ou plano de descoberta
- curso alvo
- viewports
- rubrica ativa
- destino de evidencias

## Checklist

- [ ] Resolver escopo
- [ ] Validar manifest
- [ ] Preparar navegador/perfil
- [ ] Revisar paginas em ordem
- [ ] Capturar pacote de pagina por pagina
- [ ] Aplicar verificacoes de carregamento
- [ ] Aplicar verificacoes de navegacao
- [ ] Aplicar verificacoes de layout
- [ ] Aplicar verificacoes de imagem
- [ ] Aplicar verificacoes de video quando existir
- [ ] Registrar itens de revisao humana
- [ ] Gerar relatorio
- [ ] Auditoria Final

## Saida

- relatorio de curso
- cobertura
- achados
- evidencias
- pacotes de pagina
- estado sugerido
